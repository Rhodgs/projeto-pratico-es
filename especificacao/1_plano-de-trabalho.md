## 1. INTRODUÇÃO

### 1.1 Objeto
O sistema **Jornada Verde** é uma plataforma gamificada para conscientização ambiental. O foco principal é uma **aplicação voltada para dispositivos móveis (celular)**, permitindo que o usuário interaja com o meio ambiente de forma prática.

### 1.2 Motivação, Justificativa e Oportunidade
O projeto utiliza a tecnologia para engajar jovens estudantes em práticas sustentáveis através de desafios e recompensas, resolvendo a falta de interatividade no ensino ambiental tradicional.

### 1.3 Caracterização do Projeto
#### 1.3.1 Classe
| Classe | Detalhamento |
| :--- | :--- |
| Aplicativo Móvel | O sistema será um app para ajudar na ajuda na consciencia ambiental através dos jovens |

#### 1.3.2 Enquadrabilidade
| Enquadrabilidade | Detalhamento |
| :--- | :--- |
| Software Educacional | Focado no aprendizado e conscientização ambiental escolar. |
| Software Proprietário | O sistema não será de código aberto nesta fase inicial. |

## 2. INFORMAÇÕES GERAIS

### 2.1 Escopo Geral
O projeto visa desenvolver uma solução mobile que transforme a educação ambiental em uma jornada interativa e gamificada para alunos e professores.

#### 2.1.1 Escopo Específico
* Desenvolvimento de sistema de gamificação (XP, medalhas e rankings).
* Registro de ações sustentáveis via fotos e validação de impacto.
* Painel administrativo para professores gerenciarem turmas.

#### 2.1.2 Escopo Negativo
* Não haverá integração obrigatória com redes sociais.
* O sistema não terá versão para Desktop (Windows/Linux).
* Não haverá suporte para múltiplos idiomas nesta versão.

### 2.2 Ambiente de Desenvolvimento
| Componente | Tecnologia/Ferramenta |
| :--- | :--- |
| Metodologia | Scrum com sprints |
| Gerenciamento de Backlog | GitHub Projects |
| Repositório de Código | GitHub |
| Modelagem de Software | Draw.io / Figma |
| Modelagem de Personas | HubSpot Make My Persona |

### 2.3 Características Inovadoras do Projeto
* **Gamificação de Impacto Real:** Transformação de ações físicas (reciclagem) em recompensas digitais.
* **Acessibilidade Móvel:** Foco em levar a educação ambiental para fora da sala de aula.

### 2.4 Resultados Esperados
* Engajamento de pelo menos 80% dos alunos nas missões propostas.
* Centralização de locais de descarte correto em uma interface única.
* Geração de relatórios de impacto ambiental para a coordenação escolar.
